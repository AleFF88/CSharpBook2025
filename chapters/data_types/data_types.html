Ссылочные и значимые типы данных в C#

CLR поддерживает две разновидности типов: ссылочные и значимые. Большинство типов являются ссылочными, однако зачастую мы используем именно значимые типы.

Переменные значимого типа хранятся в стеке или непосредственно внутри объектов, если являются полями. Переменная такого типа хранит само значение, а не ссылку на него. При копировании создаётся полная независимая копия.

В переменных ссылочного типа хранятся ссылки на объекты. При создании переменной выделяется память в управляемой куче, а оператор возвращает адрес этой памяти.

При копировании ссылочной переменной копируется не объект, а ссылка. Поэтому любые изменения через одну переменную отразятся на другой — они указывают на один и тот же объект в памяти.

Также, размещение в управляемой куче требует ресурсов, и после завершения использования такие объекты удаляются сборщиком мусора.

Для повышения производительности при работе с простыми данными используются значимые типы. Обычно они размещаются в стеке и не требуют сборки мусора, что снижает нагрузку на систему.

Значимый тип не может быть базовым для других типов — ни значимых, ни ссылочных.

Примитивные значимые типы

  • bool — логический тип (true или false)
  • byte — 8-битное беззнаковое целое
  • sbyte — 8-битное знаковое целое
  • char — символ Unicode (16 бит)
  • short — 16-битное знаковое целое
  • ushort — 16-битное беззнаковое целое
  • int — 32-битное знаковое целое
  • uint — 32-битное беззнаковое целое
  • long — 64-битное знаковое целое
  • ulong — 64-битное беззнаковое целое
  • float — 32-битное число с плавающей точкой
  • double — 64-битное число с плавающей точкой

Другие значимые типы

  • enum — перечисления (набор констант)
  • struct — структуры, объединяющие несколько значений

Ссылочные типы

  • object — базовый тип для всех в C#
  • class — классы с состоянием и поведением
  • interface — контракты (методы и свойства без реализации)
  • delegate — делегаты, указывающие на методы
  • array — массивы
  • string — строки (неизменяемые)
